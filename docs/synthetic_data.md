# Synthetic Data Policy

Synthetic data in this repository is generated solely for demonstration,
education, and system design purposes.

## Principles
- Clearly labeled (is_synthetic = true)
- Derived from public ranges and constraints
- No patient-level simulation
- No representation as real commercial performance

## Intended Use
- Metric modeling
- Scenario analysis
- AI evaluation
- Decision workflow design

## Not Intended For
- Financial reporting
- Medical conclusions
- Regulatory decisions
